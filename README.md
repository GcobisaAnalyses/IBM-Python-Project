# IBM Python Project - Stock Analysis

## Project Overview
Analysis of Tesla and GameStop stock data using Python, including:
- Stock data extraction with yfinance
- Revenue data web scraping
- Data visualization with matplotlib
- Comparative financial analysis

## Files
- `stock_analysis.ipynb` - Main Jupyter notebook with complete analysis
- `screenshots/` - Results and outputs for each question
- `requirements.txt` - Python dependencies

## Technologies Used
- Python
- yfinance
- BeautifulSoup
- pandas
- matplotlib
- Jupyter Notebook

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Open Jupyter: `jupyter lab`
3. Run the notebook
